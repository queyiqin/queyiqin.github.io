#利用 Hexo + Node.js + GitHub 搭建自己的博客

☆ 本文字数：2082

Windows 下具体步骤：

安装 Node.js

安装 Git

安装 Hexo:

启动 Git 界面如下：

  • 全局安装 : npm install hexo -g

  • 初始化：例如在 d 盘中创建文件夹 MyBlog，在文件夹中启动 Git Bash 输入命令 hexo init

  • 配置安装依赖包： npm install

  • 生成部署文件： hexo generate

  • 启动本地服务： hexo server

  • 最后在 http://localhost:4000/ 查看本地效果，如下：

注意：安装过程指向的是国外，如遇安装失败，把安装镜像转向淘宝
npm config set registry "https://registry.npm.taobao.org
用法 npm 换成 cnpm 其他地方不变。

GitHub 配置

GitHub 是一个面向开源及私有软件项目的托管平台，可以作为免费的服务器，免去了自己搭建服务器和写数据库的麻烦。使用限制推荐 1-G 以内,达到 1-G 以后会受到 GitHub 的通知邮件,上传超过 50-M 的单个文件会 Warning。

  • 注册 GitHub：

  • 登录：

  • 创建仓库： 在 https://github.com/new 页面创建自己的仓库当然记得选择 Public
    格式必须服从 username/username.github.io

  • 接下来或者秘钥,因为我们上传文件需要，需要设置在 https://github.com/settings/keys 此页面。如下图：

获取 Key：

  1. 检查本机是否存在 Key ls -al ~/.ssh 有则删除。

  2. 使用注册的邮箱获取： ssh-keygen -t rsa -C "XXXXX@qq.com"

  3. 生成秘钥： cat ~/.ssh/id_rsa.pub 如下：
    复制秘钥到 https://github.com/settings/keys 选择 New SSH key 名字可以随便，把秘钥复制上去即可。

  4. 设置名字本地用户名，邮箱（邮箱可以不是 GitHub 注册使用的）：

    git config --global user.name "<your name>"

    git config --global user.mail "<your mail>"

  5. 修改刚才创建的 Myblog 下面的 _config.yml 文件，找到 deploy: 下面内容改为(注意冒号后面加空格在填写地址)

type: git

repository: https://github.com/username/username.github.io.git

branch: master

安装 Git 包

  • npm install hexo-deployer-git --save 需要注意的是这也是外网,如不行请指向淘宝国内镜像。

  • 部署到 GitHub 上 hexo deploy

注意：第一次使用需要登录账户，如果邮箱登录失败可以使用账户名登录。

  • 成功后访问 https://username.github.com

  • 最后记住每次提交的步骤都一样：hexo clean ,hexo generate ,hexo deploy 三步。

以上是全部内容,作为程序员我觉得应该拥有自己的博客,嘿！
