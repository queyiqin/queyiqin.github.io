面向对象高级编程

一.类和实例

  • 通过 class 关键字定义

    clas Student():
    pass

s =Student()

  • 绑定属性： s.name

  • 前后两个下划线的为特殊方法

    class Student(Object):
    def __init__(self,name,score):  # 固定参数 self
        self.name = name
        self.score = score

二.访问限制

在 Python 中 class 中变量名以两个下划线开头的为 private 私有变量。

class Studnet(Object):
    def __init__(self,name,score):
        self.__name = name 
        self.__score = score

无法通过实例对象直接访问,注意是仍然可以访问

s = Student()
s._Student__name #不建议这样访问

  • 继承

  class Animal():

a = Animal()

五.类属性和实例属性

  • 类属性

    class Student(object):

六.slots

slots 用来限制实例动态添加属性

class Student(object):
    __slots__=('name','score')

除了可以绑定 name score 两个属性之外其他则无法绑定！注意
仅对当前类有效,子类无限制。

七.使用 @property

class Student(object):
   @property
   def name(self):
       return self._name
   @name.setter
   def name(self,value):
       self._name=value
