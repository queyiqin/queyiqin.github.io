跟我一起学 Python (一)

发表于 2017-09-20

为什么要学习 Python ?

Python 的应用范围非常广泛,自动化运维,web编程,人工智能,科学计算,图形界面开发等都可以使用 Python。
首选是网络应用,包括网站、后台服务等等；著名的网站 Youtube,豆瓣网就是用 Python 开发的。

话不多说,下面我们进入 Python 学习。

Python语言优缺点：

  • 第一运行速度相比 C++ ,Java 会慢一些,因为 Python 是解析型语言,代码在执行时会一行一行地翻译成CPU能理解的机器码.

  • 第二个缺点就是代码不能加密。如果要发布你的Python程序，实际上就是发布源代码。

下载安装 Python:

https://www.python.org/

安装好之后,在命令行运行 Python 启动 CPython 解释器。

也可以使用 Sublime Text 文本剪辑器：

http://www.sublimetext.com/

输入输出：

input()

print()

注解： # 号

Python 程序是大小写敏感的，如果写错了大小写，程序会报错。
Python 使用缩进来组织代码块，请务必遵守约定俗成的习惯，坚持使用 4 个空格的缩进。

整数类型：

  • 整数

  • 浮点数

  • 字符串

转义字符 \ 可以转义很多字符，比如 \n 表示换行，\t 表示制表符，字符 \ 本身也要转义，所以 \ 表示的字符就是 \。

布尔值 ：

  • and ： 全部为 True 才是 True。

  • or ： 只要一个为 True 为 True。

  • not ： 非 。

空值：

None 表示 None 不能理解为 0，因为 0 是有意义的，而 None 是一个特殊的空值。

变量:
变量名必须是大小写英文、数字和_的组合，且不能用数字开头：
如：

aa_01

常量： 全为大写字母

除法： / 浮点数结果 整数除法： //

字符串和编码：

  • 获取字符的整数表示： ord()

  • 把编码转换为对应的字符: chr()

Unicode 表示的 str 通过 encode()方法可以编码为指定的 bytes,
要把 bytes 变为 str，就需要用 decode() 方法。
要计算 str 包含多少个字符，用 len() 函数：

格式化：

  • 整数 %d

  • 字符 %s

  • 浮点数 %f

  • 十六进制 %x

List[] 和 Tuple()

  • List 是有序集合,可以随时添加，删除其中的元素。

追加、插入、删除函数：

append()
insert()
pop()

  • Tuple 一旦初始化就无法修改,所以安全性较高。

  • TUple 缺陷就是初始化时必须要确定下来。

条件判断语句

if else elif

if <条件判断1>:
执行1

elif <条件判断2>:
执行2

else:
执行3

循环语句语句

  • 一种是for…in循环

    for x in range(100):

    print(x)     #  输出  1 -100整数

  • while 只要条件满足，就不断循环，条件不满足时退出循环。

使用 Dict{} 和 Set()

1.dict 字典,以键值对的方式 (key-value),Key 不重复。

  • 优点 ： 具有极快的查找速度。

  • in 检查 Key 是否存在字典里。

  • get(Key) 根据 Key 获取值。

  • pop(Key) 删除 Key。

重点：list dict 性能比较：

  • dict 查找和插入速度极快，不会随着 Key 的增加而变慢，dict占用内存大。

  • list 查找和插入的时间随着元素的增加而增加。

set:

  1. set是无序的集合。set 和 dict 类似,也是一组 Key的集合，set 中 Key 不重复。

  • 创建 set 需要提供 list 作为集合。

  • 添加 set 元素使用 add (Key) 删除元素使用 remove(Key)

注意： 可以使用 set 去除重复元素、合并相同元素，交集 & ,并集 |
